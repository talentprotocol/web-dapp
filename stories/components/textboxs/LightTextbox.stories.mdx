# Buttons

Custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

## Examples

There are 6 currently supported types, `primary`, `info`, `success`, `warning`, `danger` and `secondary`. We use react-bootstrap button underneath so we accept any prop they use.
